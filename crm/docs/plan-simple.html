CRM Work Plan — HK Builders

Next phase. Updated 24 June 2026.

1. Lead Details (Before Closing)

  • Name (optional)
  • Address (optional)
  • Contact No. (required — main identifier)
  • Project
  • Investment Amount
  • Unit
  • Category
  • Agent Name
  • Status
  • Lead From (source)

All fields optional except contact number. Sometimes we only have a phone number, so the form must accept a lead with just that.

2. Client Details (After Closing)

  • Name
  • Address
  • Unit No.
  • Category
  • Project
  • Block
  • Contact No.
  • Booking Amount
  • Agent Name
  • Lead From

When a lead is marked "Won", it becomes a Client record with these booking details added.

3. Agent Details (When Creating an Agent)

  • Name
  • Email Address
  • Contact No.
  • Address
  • CNIC No.
  • Guardian No.
  • Designation

4. Agent Detail Page

  • Add a "View Details" button on each agent in the Agents page.
  • Add a "View Details" button on each agent in the Reports page.
  • Show how many leads/clients the agent contacted.
  • Show count for every status: New, Contacted, Qualified, Proposal, Negotiation, Won, Lost, Dead.
  • Show grouped totals: interested, not interested, qualified, rejected, all closings.
  • Add a month filter for the stats.
  • Make the agent report printable.

5. Agents Page Stats

  • Add a stats bar at the top: number of agents, active, suspended, total leads, total closings, average conversion.

6. Teams & Sales Managers

  • List of sales managers, each leading a team.
  • Name teams: Team A, Team B, Team C — each with a team leader and agents under them.
  • Put this on a separate page.
  • Separate stats page per team: response, sales, and earnings of each team.

7. Accounts Panel (New)

Salaries & Commissions

  • See each agent's salary and commission.
  • Mark salary/commission as paid or not paid.

Expenses & Income

  • Marketing expense, office expense, other costs.
  • Total income with date-wise filters.

Client Payments & Immature Files

  • Track client payments (e.g. down payment 6 lakh, file closed at 3 lakh = immature file).
  • On an immature file, agent commission/rebate stays pending.
  • Show the reason payment is due: no funds, or file is immature.

8. Printed Reports

  • Print each agent's report professionally.
  • Print all reports (full company reports).
  • Team reports, salary sheets, and accounts summaries — all printable.
