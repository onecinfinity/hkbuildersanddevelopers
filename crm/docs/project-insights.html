Executive SummaryThe big picture

The HK Builders CRM is a secure, custom-built sales platform that turns raw leads into booked clients — and now grows into a full operations system covering teams, agent performance, and company accounts.

The first release delivered a complete lead-to-deal pipeline: admins add leads (manually or by CSV), agents claim them from a shared pool, work them through pipeline stages, and close them. Everything is tracked, audited, and reportable. It is live, mobile-responsive, and hosted on the company domain.

The next phase expands the CRM from a lead tool into a business management platform — richer lead & client records, detailed agent profiles, a team/sales-manager hierarchy, a full accounts module (salaries, commissions, expenses, immature-file tracking), and professional printable reports.

Security FoundationBuilt in, not bolted on

🔐Access & Identity

  • Bcrypt password hashing (cost 12)
  • Role-based access — admin vs agent
  • Session fixation prevention on login
  • "Remember me" — hashed, rotating tokens

🛡Attack Defence

  • CSRF token on every form
  • Brute-force lockout (5 tries / 15 min)
  • All output escaped — no XSS
  • Prepared statements — no SQL injection

📝Accountability

  • Audit log of every sensitive action
  • Login success / fail / block tracking
  • Full activity timeline per lead
  • Soft delete — nothing is truly lost

⚙Hardening

  • Secure HTTP headers set
  • Server fingerprint stripped
  • Upload validation (CSV only, size capped)
  • Errors hidden in production

The signature feature — atomic lead claiming

When agents share a lead pool, two agents could grab the same lead at the same instant. The system prevents this at the database level: a claim only succeeds if the lead is still unassigned, so a lead can never be double-claimed. Fair, race-proof, and invisible to the user.

The immature-file concept

If a client's down payment is, say, 6 lakh but the file closes at 3 lakh, it's an immature file. The agent's rebate/commission stays pending — and the system records why the payment is due:

  • The file is immature (under-funded), or
  • We currently don't have the funds to pay out.
  • Professional printing — every agent report, team report, and full company report prints cleanly on HK Builders letterhead.
  • Agents page stats — a summary bar showing number of agents, active/suspended, total leads & closings.
